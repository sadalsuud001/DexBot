# DexBot
DexBot is a robot fleet operations platform that fuses three layers in a single product: a community + content layer; a device lifecycle layer; and an AI-powered fleet orchestration layer. Frontend is React 19 + Vite 7 + TypeScript; backend is Rust/Axum for core services with Go; deployed on AWS ECS with PostgreSQL, TimescaleDB, Kafka, Temporal.

frontend: under apps\web 
npm run dev
backend: under apps\api
